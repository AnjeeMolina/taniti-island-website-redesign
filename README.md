# 🌴 Taniti Island Tourism Website Redesign

## 📌 Overview
This project is a UI/UX redesign of the Taniti Island tourism website. The goal was to improve usability, navigation, and overall user experience to create a modern, organized, and mobile-friendly design that encourages tourism.

---

## 🎯 Problem
The original website had several usability issues:
- Confusing navigation
- All content placed on a single page
- Outdated design
- Lack of booking information and structure

These issues made it difficult for users to explore the island and plan their visit.

---

## 🎯 Goal
Design a user-friendly tourism website that:
- Improves navigation and structure
- Organizes information into multiple pages
- Enhances visual design and readability
- Supports mobile-friendly use
- Encourages users to explore and engage with the site

---

## 👥 Target Users
- Tourists planning vacations
- First-time visitors
- Users looking for activities, lodging, and travel information

---

## 🧠 My Approach
- Analyzed the problems in the existing website
- Broke content into clear sections (activities, lodging, dining, transportation)
- Designed a structured multi-page layout
- Focused on usability and user flow
- Applied visual hierarchy and consistency

---

## ✏️ Design Process
- Created page layouts and structure
- Designed navigation and content flow
- Refined layout, spacing, and visual hierarchy

---

## 💡 Key Improvements
- Clear and intuitive navigation
- Organized content across multiple pages
- Improved visual hierarchy
- More modern and engaging design
- Better overall user experience

---

## 📚 What I Learned
- How to identify usability issues
- Designing with user experience in mind
- Structuring content effectively
- Improving layout and readability
